# 番茄小说短故事创作指南

基于《番茄小说短故事写作风格研究报告》提炼，适用于番茄平台的短故事创作。平台定义的短故事范围为 6000-80000 字，其中 10000-30000 字最适合新稿起步。

## 一、平台特征

### 核心阅读习惯
- 碎片化阅读明显，前三章必须有吸引力。
- 平台偏好强情绪、快节奏、高密度爽点。
- 前 3000 字直接影响留存和推荐表现，不能慢热。
- 听书渗透率高，文风要口语化、短句化、易朗读。

### 番茄短故事硬指标

| 节点 | 必须完成的任务 |
|------|----------------|
| 前 300 字 | 抛出核心冲突或异常事件 |
| 前 1000 字 | 交代主角困境、欲望和短期目标 |
| 前 3000 字 | 完成黄金三章，建立主线吸引力 |
| 每 500 字左右 | 埋一个小反转或小信息差 |
| 每 2000 字 | 设置一个小高潮 |
| 每章结尾 | 必须留钩子 |
| 每 1 万字 | 安排一次大反转或格局升级 |

### 黄金三章模板

| 章节 | 核心任务 | 写法重点 |
|------|----------|----------|
| 第一章 | 立人设 + 抛悬念 | 300 字内给冲突，500 字内明确困境 |
| 第二章 | 强化冲突 | 升级矛盾，引入关键人物、线索或金手指 |
| 第三章 | 反转打脸 | 完成第一次逆袭，确立爽文基调，结尾再抛新钩子 |

## 二、核心创作原则

### 1. 开篇先炸，不要铺垫
- 优先使用危机、背叛、倒计时、身份反差、金手指落地这类高能开局。
- 开头第一屏就要让读者知道“主角遇到了什么大事”。
- 不要先讲背景，再讲冲突；正确顺序是先冲突，再补背景。

### 2. 节奏要像情绪过山车
- 常用情绪线：委屈 -> 愤怒 -> 压抑 -> 反击 -> 解气。
- 虐点之后要尽快给回报，复仇或打脸最好不要拖过 3 章。
- 每章都要有推进，不能出现“信息没变、局势没变、情绪没变”的平章。

### 3. 人物要标签化，但不能扁平
- 主角最好能被三个词概括，如“落魄、嘴硬、能反杀”。
- 用反差制造记忆点，如“窝囊赘婿其实手握底牌”“冷脸医生偏偏怕猫”。
- 人设靠行动和对白立住，不靠长段介绍。

### 4. 爽点要密，反转要有因果
- 常见爽点：打脸、逆袭、掉马、复仇、身份曝光、情感释放。
- 反转必须提前埋线，做到意料之外、情理之中。
- 老套路可以用，但必须加一层新元素或新组合。

### 5. 文字必须适配听书
- 短句优先，少用过长复句和生僻词。
- 多用对白和动作推动情节，少写大段解释。
- 标点要清晰，方便语音停顿和情绪起伏。

## 三、类型文写法

### 都市言情
- 开局优先写离婚、背叛、重生、当众羞辱、关系破裂等高情绪场景。
- 女主线更适合“清醒后主动反击”，不要长期被动承受。
- 常用组合：追妻火葬场、重生复仇、先婚后爱、马甲掉落。
- CP 设计要有天然冲突和反差，关系线最好自带拉扯感。

### 玄幻修仙
- 世界观讲究精简高效，首章尽快交代时间、地点、核心危机。
- 修炼体系要标准清晰，等级最好可视化，方便读者感受成长。
- 金手指遵循“强、限、长、贴”：功能强，限制硬，成长线长，和主线贴合。
- 战斗描写要简洁利落，几回合定胜负，少做冗长特效描写。

### 悬疑推理
- 开篇核心是认知冲突，让读者立刻意识到“这件事不对劲”。
- 信息控制尽量和主角同步，增强代入式解谜体验。
- 推荐结构：开篇悬念 -> 3 次小反转 -> 1 次大反转 -> 闭环收尾。
- 恐怖感更多依靠细节和异常感，不靠直白血腥堆砌。
- 配角尽量全员可疑，每个人都有秘密或遮掩。

## 四、标题与钩子策略

### 标题方向
- 优先突出冲突、反差、悬念、倒计时、掉马、重生、离婚、系统等强卖点。
- 标题不要过于文艺含混，要让读者一眼知道爽点或疑点。
- 可直接使用“身份/处境 + 核心冲突”结构。

### 一章三钩
- 章中钩：中段抛出新信息、改口、异动或小秘密。
- 章末大钩子：停在身份将暴露、真相将揭开、危险将发生的瞬间。
- 主线悬钩子：全书长期悬念，如身世、幕后黑手、真正规则。

## 五、番茄适配检查清单

- 是否符合 6000-80000 字范围，且优先控制在 10000-30000 字？
- 前 300 字是否出现强冲突，前三章是否持续抓人？
- 是否做到每章有钩子、每 2000 字有小高潮？
- 主角是否有清晰欲望、记忆点和可执行的反击路径？
- 金手指、规则、悬念是否有限制、有伏笔、能回收？
- 语言是否口语化、短句化，适合听书？
- 标题、开篇、章节结尾是否都在放大卖点？

## 六、常见误区

- 开篇先讲背景，迟迟不进冲突。
- 人物太多，前三章就把读者写晕。
- 节奏只有压抑没有回报，爽点来得太晚。
- 金手指过强且无代价，直接把悬念写死。
- 反转没有铺垫，只靠强行揭晓。
- 句子太长、描写太满，听书时不顺口。

---

使用建议：当用户选择番茄短故事风格时，先按本指南约束开篇、节奏、钩子和语言，再结合具体题材补充世界观、情感线或谜题设计。
