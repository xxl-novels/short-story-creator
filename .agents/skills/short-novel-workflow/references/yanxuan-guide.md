# 知乎盐选小说创作指南

基于知乎盐选平台标准，适用于1-3万字短篇小说的创作规范。

## 一、平台特征

### 篇幅与结构
```
篇幅一到三万字，短小精悍成本低；
主线贯穿支线明，反转悬念要跟上；
节奏明快别拖沓，跌宕起伏才精彩；
人物饱满有成长，引发共鸣有思考；
通俗流畅口语化，简洁明了排版清；
```

### 盐选结构硬性指标

| 字数节点 | 必须完成的任务 |
|---------|--------------|
| 前200字 | 拎主线，找看点 |
| 前1000字 | 明确主角欲望+动机 |
| 前3000字 | 预埋悬念（钩子） |
| 前1/3或1/2 | 设置大转折或小高潮 |

### 发展阶段要求
- **发展阶段**：每千字至少一个看点，冲突要激烈
- **高潮部分**：全文最大转折点，承上启下
- **结构公式**：开头抛出悬念 → 发展强化悬念 → 高潮解开悬念

## 二、类型文写作指南

### 1. 世情文

**定义**：以现实社会题材为背景，描写家庭伦理、婆媳婚姻、人情世态的故事。

**爽点公式**：前期主角实惨 → 后期逆袭/反派被打脸

**热门组合**：
- 世情+打脸爽文（如《逝水》- 父子火葬场）
- 世情+大女主（如《真假爱意》）
- 世情+古言（如《点燃星火》）
- 世情+闺蜜同嫁（如《和闺蜜一起穿进救赎文》）

**人物塑造要点**：
- 主角人设：极端反转
- 反派人设：极端坏
- 代入感三技巧：心理描写、对话描写、环境描写

### 2. 悬疑文

**生活悬疑定义**：描写日常生活、不涉及刑事案件、从细微处挖掘谜团。

**选题方向**：
| 类型 | 案例参考 |
|-----|---------|
| 悬疑+职场 | 《6个说谎的大学生》 |
| 悬疑+家庭 | 遗产、出轨、亲戚较量 |
| 悬疑+脑洞 | 时空穿越、特殊能力 |
| 悬疑+日常 | 校园、火车站、医院 |

**人物设计4要素**：
1. 生理特征（性别、年龄、外貌、教育、职业）
2. 性格特点（臭脾气、坏习惯）
3. 弱点和缺陷（不要十全十美）
4. 特别的语言习惯（方言加分）

**线索布置技巧**：
- 使用不同类型的线索（证词、日记、回忆）
- 设置至少一个误导性线索（案中案、顶包、叙述性诡计）
- 将线索情感化，与角色经历联系

### 3. 古言文

**苏点**：男女主情感暧昧极限拉扯感
**虐点**：身份地位对立、遗憾错过、宿命感冲突

**6大类型**：
1. 古言追妻文（女性主动+打脸渣男）
2. 古言甜宠文（先婚后爱、情感拉扯）
3. 古言权谋文（大女主+家国情怀）
4. 古言脑洞文（动物主角、快穿、金手指）
5. 古言玄幻文（修炼晋级推动情节）
6. 古言同人文（公版内容二次创作）

**提升代入感5方法**：
1. 开篇强冲突（情绪进入最快）
2. 主角人设真实可感
3. 多用细节描写
4. 动机合理化（人物是时代的产物）
5. 配角特色化（脸上有疤、说话结巴等记忆点）

### 4. 虐文

**人物关系**：要亲密，打破时才更痛

**虐点分类**：
| 类型 | 说明 | 适合人群 |
|-----|------|---------|
| 外力虐点 | 年龄、家庭矛盾、经济状况、地位差异 | 新手 |
| 内因虐点 | 宗教信仰、生活方式、欲望动机、性格冲突 | 进阶 |

**情绪节奏**：先抑后扬
- 30%处：体现开始释放情绪的信号
- 60%处：释放文章高潮

## 三、核心创作原则

### 开篇要求
1. 一句话能提炼出核心亮点
2. 开篇干脆不拖沓，快速抛出故事基调
3. 把最吸引人的情节、主角最核心的欲望提炼出来

### 中段要求
1. **核心悬念**：开篇抛出，用钩子勾着读者
2. **人物张力**：设置完整成长路径
3. **文稿创意**："给我相同的东西，但要不一样"——用新视角解决旧元素

### 结局要求
1. 形成完整闭环（危机解除后的轻松感）
2. 人物结局交代清楚
3. 升华主题，耐人寻味

## 四、调动读者情绪

### 四个维度
1. **人物塑造**：人设真实（美强惨）、有成长线、复杂人际关系
2. **情节设定**：增加读者共鸣事件、将日常事件极致化
3. **场景设置**：明确目的、选取3-4个重点细节、符合观察逻辑
4. **悬念冲突**：紧张感、期待感、反差感、情感冲突

### 常见共鸣点
- 家庭关系控制（如《茧爱》）
- 职场困境（如《过河拆桥后公司把自己裁没了》）
- 婆媳关系、感情纠葛、重男轻女

## 五、7+3大纲法

### 7个步骤
1. **弱点与需求**：故事开头设置主角弱点，需满足内在需求
2. **欲望**：主角想得到的东西或目标
3. **对手**：与主角竞争同一个目标，有深层冲突
4. **计划**：如何击败对手或达成目标
5. **对决**：最后冲突，决定胜负
6. **真实自我的揭露**：角色转变的终点
7. **新的平衡点**：事情恢复正常，主角发生转变

### 3元素
幽默感 + 动作 + 关系

## 六、新手常见误区

| 误区 | 解决方案 |
|-----|---------|
| 场景太平淡 | 设置悬念，透露事情并非表面如此 |
| 倒叙处理不当 | 倒叙段落构成完整场景，节奏紧凑充满冲突 |
| 面临情节岔路 | 用索引卡记录，理性分析后再决定 |
| 写作陷入苦战 | 回头删改、移动角色时空 |
| 创作脑袋短路 | 充电放松、重新体验场景、回顾初衷 |

## 七、素材来源建议

1. 社会新闻、短视频热点
2. 知乎站内爆款故事
3. 小区/学校/公司的八卦
4. 家庭矛盾、婆媳关系、重男轻女等现实议题
