# 《坠楼当晚，老公正在陪白月光》设定集

---

## 一、世界观设定

### 时间线
- **前世：** 2023年5月-2026年3月（女主死亡到男主重生）
- **重生线：** 2023年4月（死亡前一个月）

### 地点
- **江城：** 虚构一线城市
- **陆氏集团：** 陆家的商业帝国，涉及地产、金融
- **苏家：** 女主娘家，商业联姻对象
- **凶宅：** 陆景深和苏晚棠的婚房，女主坠楼地点

---

## 二、人物详细设定

### 陆景深（男主）

**基础信息**
- 年龄：28岁
- 身份：陆氏集团继承人
- 性格：冷峻、傲慢、极度自我
- 弱点：对母亲有盲目信任，对白月光有青春滤镜

**前世轨迹**
```
商业联姻 → 婚礼出轨 → 三年冷暴力 → 女主死亡 → 三年醉生梦死 → 发现真相 → 崩溃 → 重生
```

**关键记忆点**
- 婚礼当天在白月光酒店房间的时间：13:00-15:00
- 死亡当晚关机时间：20:30-次日8:00
- 17通未接电话：23:17-23:43（女主死亡前）
- 重生时间点：2023年4月1日（愚人节）

**重生后变化**
- 前期：急切弥补，以为能挽回
- 中期：发现女主也重生，发现自己参与谋杀
- 后期：赎罪式卑微，接受永远不原谅

**标志性动作**
- 前世：关机、冷笑、不耐烦
- 重生后：看手机（确认时间）、跪姿、流泪

---

### 苏晚棠（女主）

**基础信息**
- 年龄：25岁
- 身份：苏家独女，设计专业
- 性格：前世隐忍，重生后冷静、决绝
- 能力：设计天赋、商业嗅觉

**前世轨迹**
```
暗恋陆景深 → 商业联姻 → 发现出轨 → 隐忍保胎 → 跟踪调查 → 被追杀 → 17通电话 → 坠楼
```

**死亡当晚时间线**
```
20:00 发现陆母挪用公款证据
20:15 给陆景深打电话（未接）
20:30 被跟踪，进入旧楼
23:17 开始连续打电话（17通）
23:43 坠楼
```

**重生时间点**：2023年3月29日（比男主早3天）

**重生后布局**
- 第1天：确认重生，收集情绪
- 第2天：开始布局，接触周牧野（假装不知情）
- 第3天：男主重生，她已做好准备

**标志性台词**
- "我不原谅你，但我给你机会证明你配被我原谅。"
- "这是你上一世欠我的，这一世我还给你。"

---

### 陆母（沈如玉）

**基础信息**
- 年龄：55岁
- 身份：陆氏集团实际掌控者（董事长）
- 性格：控制狂、冷酷、为达目的不择手段

**罪行清单**
1. 挪用公款2亿（填补炒股亏损）
2. 杀害苏晚棠（推下楼）
3. 伪造不在场证明（穿儿子衣服、用儿媳手机）
4. 灭口保姆张妈

**杀人动机**
- 苏晚棠发现她挪用公款
- 苏晚棠怀孕，怕私生子分家产
- 苏晚棠死后可以嫁祸给白月光，一举两得

**标志性动作**
- 端茶（那杯安神茶）
- 冷笑"都是为了陆家"

---

### 沈念薇（白月光）

**基础信息**
- 年龄：26岁
- 身份：陆景深大学同学，小演员
- 性格：绿茶、贪婪、愚蠢

**核心秘密**
- 孩子是周牧野的（不是陆景深）
- 和周牧野合谋骗婚分财产
- 伪造短信的共犯（虽然是被利用）

**下场**
- 发现被骗后捅死周牧野
- 一尸两命（自己怀孕）
- 入狱

---

### 周牧野（男主兄弟）

**基础信息**
- 年龄：28岁
- 身份：陆景深发小，陆氏集团副总
- 性格：阴险、伪善

**核心秘密**
- 和沈念薇偷情多年
- 孩子是她的
- 帮陆母善后（处理监控、买通保姆）

**下场**
- 被沈念薇捅死
- 死前才知道孩子是自己的

## 四、关键场景

### 场景1：凶宅地下室（开篇）
- 氛围：阴冷、潮湿、灰尘
- 关键元素：监控硬盘、保姆尸体
- 情绪：男主的悔恨和恐惧

### 场景2：婚礼化妆间（回忆）
- 氛围：喜庆外表下的肮脏
- 关键元素：婚纱、手机、裸照
- 情绪：讽刺

### 场景3：死亡当晚酒店（回忆）
- 氛围：奢靡、放纵
- 关键元素：关机键、白月光的笑声
- 情绪：读者的愤怒

### 场景4：旧楼（女主死亡地点）
- 氛围：破败、黑暗
- 关键元素：17层、天台、推搡
- 情绪：绝望

### 场景5：董事会（第11章高潮）
- 氛围：压抑、紧张
- 关键元素：投影、账本、警察
- 情绪：戏剧性爆发

### 场景6：医院走廊（第16章）
- 氛围：消毒水、冰冷
- 关键元素：手术单、跪姿
- 情绪：对称复仇的爽感

### 场景7：面试房间（第18章）
- 氛围：正式、陌生
- 关键元素：简历、位置对调
- 情绪：卑微与掌控

## 五、情绪曲线

```
第一幕（恨）：读者恨男主 → 好奇真相
    ↓
第二幕（爽）：女主布局 → 读者期待复仇
    ↓
第三幕（炸）：男主自毁 → 反派下线 → 爽感爆发
    ↓
第四幕（虐+甜）：对称复仇 → 卑微追妻 → 带刺HE
```

## 七、注意事项

1. **男主前期必须够渣**，否则火葬场没说服力
2. **女主不能心软**，重生后必须清醒决绝
3. **反派下线要爽**，不能太简单也不能太拖沓
4. **HE要带刺**，不能是大团圆，必须是"有条件的和解"
