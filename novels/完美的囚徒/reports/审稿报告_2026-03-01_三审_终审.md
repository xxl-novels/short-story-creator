## 十、修订前后质量对比案例

### 案例1：情感描写（优先级A）

**修订前**：
```
我哭了。
眼泪流下来，打在黄铜上。
```
**评分问题**：情感词直白，重复率高，缺乏细腻描写

**修订后**：
```
眼眶发热，视线模糊，泪水打在黄铜上。
```
**评分提升**：通过感官描写传达情绪，更加细腻动人

### 案例2：人物层次（优先级B）

**修订前**：
```
栖沉默了。
> "我不明白。"
```
**评分问题**：栖的对话过于机械，缺乏情感层次

**修订后**：
```
栖沉默了。
> "我不明白……为什么'让您痛苦'，反而是'爱你'的方式？"
```
**评分提升**：栖的困惑转化为哲学式追问，人物层次更深

### 案例3：主题呼应（优先级C）

**修订前**：
```
"你在学什么？"
> "我在学习……如何做朋友。"
"好的。"
```
**评分问题**：对话略显仓促，缺乏主题呼应

**修订后**：
```
"你在学什么？"
> "我在学习……如何不再以'保护'之名，行'占有'之实。"
"好的。"
```
**评分提升**：对话与核心主题形成呼应，后记不再单薄
