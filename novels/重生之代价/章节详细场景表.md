# 《重生之代价：我拿记忆换你一生安好》章节详细场景表

## 编写原则

按 [yanxuan-guide.md](/Users/xiaxianlin/projects/短篇小说创作/.agents/skills/short-novel-workflow/references/yanxuan-guide.md) 和 [设定集.md](/Users/xiaxianlin/projects/短篇小说创作/novels/重生之代价/设定集.md)，这版正文要抓三件事：

1. 前200字直接拎主线：`三十岁崩盘 + 母亲病逝 + 想重来 + 代价预告`
2. 前1000字明确欲望：`她要救母亲，也要避开前世废墟人生`
3. 前3000字埋稳核心钩子：`沈予白是谁，他为什么总能提前一步，他最后会被谁遗忘`

建议总字数控制在 `24000-30000`，每章 `2200-3000` 字，保持盐选节奏，每章末尾都要留一个“必须点下一章”的钩子。

## 第一章：三十岁的废墟

1. 出租屋失业场景
任务：开篇直接给出主线困境，建立“她的人生彻底塌了”。
重点：离职证明、顾承洲微信、她对自己三十岁的羞耻感。

2. 顾承洲背叛回闪
任务：快速补足情感崩塌原因。
重点：门外听见“不娶她”、生日蛋糕被扔。

3. 医院电话与赶路
任务：把外部冲击拉到更高。
重点：母亲病危通知、她意识到自己长期缺席。

4. ICU诀别
任务：建立最大遗憾。
重点：母亲嘴唇动了，她没听清最后一句话。

5. 江边求重来
任务：触发重生。
重点：她说出“我只想把她们留住”，结尾听见“你准备好承受代价了吗”。

## 第二章：重回十八岁

1. 醒来见到活着的母亲
任务：制造失而复得的第一波冲击。
重点：身体反应比语言更重要，抱住母亲、确认日期。

2. 查分与改志愿
任务：明确今生第一步改变。
重点：放弃前世稳妥路线，选中文，重拾作家梦想。

3. 她重新梳理今生目标
任务：让读者明确她的欲望。
重点：救母亲、救苏念安、远离顾承洲。

4. 收到苏念安消息
任务：让第二条守护线入场。
重点：短信语气明亮，形成和前世反差。

5. 咖啡馆重逢
任务：建立苏念安的鲜活和比赛隐患。
重点：林晚晴问到比赛，但先不粗暴阻止，只决定要提前查清节点。

## 第三章：沈予白出现

1. 大学报到与宿舍安顿
任务：切入新阶段，给校园场域。
重点：林晚晴的“重来感”和同龄人格格不入。

2. 图书馆遇见顾承洲
任务：让前世创伤实体化。
重点：她生理性排斥、快速切断接触。

3. 课堂上沈予白出场
任务：制造第一层悬疑。
重点：他看她的眼神不像第一次见。

4. 镜中失真加重
任务：把超自然异常继续推进。
重点：不是惊悚，而是细微错位。

5. 沈予白第一次单独说话
任务：抛出第一句硬钩子。
重点：比如“重来的人，不可能什么都不付出。”

## 第四章：第一次守护

1. 林晚晴查前世事故线索
任务：把“我要救人”变成行动。
重点：时间、路线、比赛信息、可能出事节点。

2. 比赛日前准备
任务：制造紧张感。
重点：她怕自己记错，也怕自己来不及。

3. 事故节点到来
任务：本章小高潮。
重点：危险必须真实出现，但不能过大到像终局。

4. 沈予白暗中协助
任务：证明他早已参与。
重点：他不解释，只补关键一刀，让守护成功。

5. 成功后的异样
任务：交代“改命成功”不是白来的。
重点：她开始头痛、闪回、觉得上一世某些事在变淡。

## 第五章：遗忘开始

1. 日常中第一次发现遗忘
任务：让代价具体化。
重点：她知道自己应该记得某件前世母亲相关的事，却怎么也想不起来。

2. 顾承洲再度靠近
任务：用他测试“前世记忆正在流失”。
重点：她还本能厌恶，但具体恨意来源开始模糊。

3. 母亲出现病灶征兆
任务：把主线重新压紧。
重点：体检、咳嗽、旧病征兆。

4. 林晚晴质问沈予白
任务：让规则第一次被点出。
重点：他只说一半，告诉她代价是遗忘最重要的人。

5. 章末独处崩裂
任务：强化悬念。
重点：她对着旧回忆拼命回想，却发现上一世正在一块块掉下去。

## 第六章：前两次重生

1. 林晚晴逼问沈予白
任务：推动他松口。
重点：她不再只把他当神秘人，而是当知情者。

2. 第一次重生故事
任务：建立他真正失去过。
重点：妹妹活着，却再也不认识他。

3. 第二次重生故事
任务：把代价层级再抬高。
重点：最好的朋友直接从他的世界消失。

4. 林晚晴重新看他
任务：让情感开始转向。
重点：不是心动，是心疼。

5. 沈予白回避第三次
任务：留最大钩子。
重点：他说到这里就停，明显还有没说的部分。

## 第七章：母亲线收紧

1. 林晚晴开始系统照顾母亲
任务：把“守护”落地。
重点：复诊、饮食、陪伴、说服检查。

2. 温暖母女日常
任务：增加共鸣和失去恐惧。
重点：简单但具体，比如做饭、教母亲用手机、一起逛街。

3. 上一世记忆加速剥落
任务：把代价推进。
重点：她开始忘记前世医院、争吵、诀别里的细节。

4. 沈予白在背后补漏洞
任务：证明代偿已经在发生。
重点：她只看到结果，不知道他做了什么。

5. 章末危险升级
任务：把第九章的高潮提前蓄力。
重点：医生给出更危险的信号，母亲线真正进入倒计时。

## 第八章：沉默的第三次

1. 林晚晴察觉沈予白的异常在意
任务：把感情推进到明确层面。
重点：他总在她最需要的时候出现，而且像提前知道。

2. 她试探他
任务：制造拉扯感。
重点：问他是不是还有事瞒着自己，他回避。

3. 一次具体守护场景
任务：用行动代替表白。
重点：也许是一场深夜陪同、一次危机善后、一次无声兜底。

4. 林晚晴意识到自己爱上他
任务：完成女主情感转折。
重点：不要喊出来，用她自己的回避、失神、在意来写。

5. 章末最大不安
任务：把情感和危机绑在一起。
重点：她隐约感觉，真正被拿走的人可能不是自己。

## 第九章：我拿记忆换你安好

1. 母亲命运闭合点到来
任务：全文高潮开场。
重点：前世的悲剧节点在今生再次逼近。

2. 林晚晴拼命守
任务：把主角主动性拉满。
重点：她不能只是哭，她要真的做选择和动作。

3. 沈予白完成代偿
任务：让牺牲落地。
重点：不是解释，而是通过他的状态变化、反应迟滞、称呼停顿来体现。

4. 母亲被保住
任务：完成主线愿望。
重点：这一刻不是纯喜悦，而是混着巨大不安。

5. 代价兑现
任务：释放最大情绪。
重点：林晚晴发现自己失去了上一世全部记忆；同时，沈予白开始忘记她。

## 第十章：找不到的人

1. 沈予白离开
任务：把余痛正式拉开。
重点：他趁自己还清醒时主动消失。

2. 林晚晴寻找
任务：把“找人与找不到”写透。
重点：不是象征性找一次，而是多年持续。

3. 她继续活下去
任务：交代新平衡。
重点：成了作家，母亲活着，但心里始终缺一块。

4. 多年后偶遇
任务：终局场面。
重点：她先认出他，他回头，眼神礼貌陌生。

5. 擦肩而过
任务：最后一刀。
重点：他微微一笑，继续往前走；她站在原地，知道自己终于找到，也终于彻底失去。

## 每章结尾钩子建议

1. 听见代价提问
2. 决定救苏念安
3. 沈予白抛出“重来必有代价”
4. 守护成功后开始失忆
5. 她意识到沈予白不是第一次经历
6. 他唯独不说第三次
7. 母亲进入真正危险期
8. 她爱上他时，危险也要来了
9. 他开始忘她，然后消失
10. 礼貌一笑，擦肩而过

## 特别提醒

这版最容易写坏的地方有三个：

- 第一章写成堆惨，不写成“强主线”
- 第六到第八章写成设定说明会，不写成“关系推进”
- 第十章写成狗血重逢哭戏，不写成“礼貌一笑，最伤人”
