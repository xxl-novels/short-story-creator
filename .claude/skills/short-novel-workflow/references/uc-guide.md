# UC故事会小说创作指南

基于UC故事会平台标准，适用于1-5万字短篇小说的创作规范。

## 一、平台特征

### 篇幅与结构
```
篇幅一到五万字，短小精悍节奏快；
主线贯穿冲突强，惊险刺激要跟上；
三秒法则抓眼球，跌宕起伏才精彩；
人物接地气真实，引发共鸣有爽感；
通俗口语网络化，简洁明了排版清；
```

### UC结构硬性指标

| 字数节点 | 必须完成的任务 |
|---------|--------------|
| 前3句话 | 建立吸引力（三秒法则） |
| 前200字 | 拎主线，找看点，建立冲突 |
| 前1000字 | 明确主角欲望+动机，进入核心情节 |
| 前3000字 | 预埋悬念（钩子），情节开始推进 |
| 前1/3 | 设置大转折或小高潮，刺激升级 |

### 发展阶段要求
- **发展阶段**：每千字至少一个刺激点，冲突要激烈，节奏明快
- **高潮部分**：全文最大转折点，承上启下，惊险刺激
- **结构公式**：开头强冲突 → 发展强化刺激 → 高潮爆发解决

## 二、类型文写作指南

### 1. 世情文（UC核心品类）

**定义**：以现实社会题材为背景，描写家庭伦理、婆媳婚姻、人情世态、出轨复仇的故事。

**爽点公式**：前期主角受委屈/被绿/被背叛 → 后期逆袭复仇/打脸渣男/获得补偿

**UC热门组合**：
- 世情+出轨复仇（如：发现妻子出轨，联合小三反杀）
- 世情+绿帽复仇（如：被戴绿帽后的强势反击）
- 世情+财产争夺（如：亲戚算计、遗产纠纷）
- 世情+性擦边（如：真实经历改编、猎奇情感故事）

**人物塑造要点**：
- 主角人设：普通人但有情义，有仇必报
- 反派人设：极端坏，现实感强（渣男、恶婆婆、贪心亲戚）
- 代入感三技巧：口语对话、心理活动、环境细节

### 2. 悬疑文

**UC悬疑定义**：反转悬疑、都市情感类，强调情节刺激和意外结局。

**选题方向**：
| 类型 | 案例参考 |
|-----|---------|
| 悬疑+出轨 | 妻子神秘死亡，丈夫是凶手？ |
| 悬疑+遗产 | 家族争夺遗产，每个人都有秘密 |
| 悬疑+复仇 | 精心策划多年的复仇大计 |
| 悬疑+真实案件 | 改编自真实案件，猎奇性强 |

**人物设计4要素**：
1. 生理特征（性别、年龄、职业，要接地气）
2. 性格特点（有脾气、有弱点、真实可信）
3. 秘密和动机（每个人都有不可告人的秘密）
4. 语言习惯（口语化、方言加分）

**线索布置技巧**：
- 开篇即设悬念，吸引读者继续阅读
- 多层反转，结局出人意料
- 节奏紧凑，减少铺垫，直接进入核心冲突

### 3. 玄幻/武侠文

**UC特点**：男性向冒险刺激内容，强调打斗场面和升级快感。

**6大类型**：
1. 废柴逆袭流（从底层崛起的爽快）
2. 强者归来流（王者回归碾压一切）
3. 复仇打脸流（有仇必报，十倍奉还）
4. 奇遇升级流（获得金手指，快速变强）
5. 江湖恩怨流（武侠世界的快意恩仇）
6. 都市异能流（现代背景下的超能力）

**提升代入感5方法**：
1. 开篇强冲突（打斗、追杀、危机）
2. 主角人设要有血性，有情有义
3. 多用动作和场面描写
4. 升级体系清晰，爽点明确
5. 反派够嚣张，打脸够痛快

### 4. 言情文（女频向）

**UC特点**：虐恋、追妻火葬场、白月光等类型。

**虐点分类**：
| 类型 | 说明 | 适合人群 |
|-----|------|---------|
| 外力虐点 | 家庭反对、身份差距、误会分离 | 新手 |
| 内因虐点 | 性格冲突、自尊倔强、错过时机 | 进阶 |

**情绪节奏**：先抑后扬
- 30%处：女主开始觉醒或男主开始后悔
- 60%处：追妻火葬场或感情爆发

## 三、核心创作原则

### 开篇要求
1. **三秒法则**：前三句话必须抓住读者
2. 直接切入冲突，减少背景铺垫
3. 把最刺激的情节、最核心的矛盾直接摆出来

**开篇公式**：
```
异常事件/惊人发现 + 主角反应 + 悬念钩子
例：
"我出差提前回来，发现卧室的床单换了。妻子说是一个人睡的，但我闻到了陌生的古龙水味。"
```

### 中段要求
1. **核心刺激**：情节持续推进，减少水分
2. **人物张力**：人物关系紧张，矛盾不断升级
3. **文稿创意**："给我熟悉的东西，但要更刺激"——用强冲突重新包装旧元素

### 结局要求
1. 形成完整闭环（问题解决，爽感释放）
2. 反派得到惩罚，主角获得补偿/胜利
3. 结局干脆，不拖沓，有余味即可

## 四、调动读者情绪

### 四个维度
1. **人物塑造**：人设真实接地气（有情有义的普通人）、有成长线
2. **情节设定**：增加读者共鸣事件、将日常事件极致化（如：普通出轨变成精心复仇）
3. **场景设置**：明确目的、选取3-4个重点细节、符合观察逻辑
4. **悬念冲突**：紧张感、期待感、反差感、情感冲突

### UC常见共鸣点
- 被出轨/被背叛后的复仇（如：《七年时分》）
- 亲戚算计、财产纠纷
- 婆媳矛盾、家庭伦理
- 职场被欺负后的逆袭

## 五、标题与导语策略

### 标题公式

| 类型 | 公式 | 案例 |
|-----|------|------|
| 悬念型 | 异常事件+疑问 | 《我出差回来，发现妻子换了床单》 |
| 冲突型 | 极端情况+主角反应 | 《被戴绿帽后，我笑了》 |
| 数字型 | 数字+猎奇内容 | 《结婚7年，我发现了妻子的3个秘密》 |
| 情绪型 | 强烈情绪+事件 | 《别惹老实人：我的复仇计划》 |

### 导语写法
- 直接抛出核心冲突
- 制造悬念，引发好奇
- 控制在50字以内，信息密度高

## 六、7+3大纲法（UC版）

### 7个步骤
1. **弱点与需求**：故事开头设置主角弱点或困境（被出轨、被欺负）
2. **欲望**：主角想要复仇/获得补偿/改变命运
3. **对手**：反派嚣张跋扈，不断挑衅主角
4. **计划**：主角如何策划反击或达成目标
5. **对决**：最后冲突，决定胜负，高潮爆发
6. **转折与揭露**：真相大白或局势逆转
7. **新的平衡点**：反派受到惩罚，主角获得胜利

### 3元素
刺激感 + 动作/冲突 + 现实感

## 七、UC风与知乎风对比

| 维度 | UC风 | 知乎风 |
|------|------|--------|
| **受众** | 18-60岁男性，二三线城市 | 16-25岁女性，文化程度较高 |
| **叙事视角** | 第三人称全知为主 | 第一人称代入感强 |
| **语言风格** | 通俗接地气、口语化 | 精致严谨、注重文字表现力 |
| **情节特点** | 惊险刺激、冒险打斗 | 逻辑严谨、反转悬念 |
| **题材偏好** | 世情+擦边、出轨复仇 | 虐恋、甜宠、重生 |
| **开篇要求** | 三秒法则，强冲突 | 200字拎主线，找看点 |
| **创作心态** | 追求普适性和大众吸引力 | 追求质量和口碑 |

## 八、新手常见误区

| 误区 | 解决方案 |
|-----|---------|
| 铺垫太长 | 直接切入冲突，背景后补 |
| 语言太文艺 | 用口语化表达，通俗易懂 |
| 节奏太慢 | 每千字必须有刺激点 |
| 人物太完美 | 给主角设置弱点和困境 |
| 反派太弱 | 反派要嚣张，打脸才爽 |
| 结局太啰嗦 | 问题解决后快速收尾 |

## 九、素材来源建议

1. **社会新闻**：出轨、遗产纠纷、刑事案件改编
2. **短视频热点**：抖音/快手上的情感纠纷、家庭矛盾
3. **身边八卦**：小区/公司的真实事件加工
4. **传统类型**：经典爽文套路+现实背景包装
5. **UC必读榜**：研究榜单作品，把握平台风向
